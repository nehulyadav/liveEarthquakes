Simple steps to run:

For Mac Users:

1. Clone the project to your local computer.
2. Open Terminal and cd into webgl-globe-master 2.
3. Type python -m SimpleHTTPServer 8000
4. Open your browser and run localhost:8000/globe/
5. Take tour of the globe!
